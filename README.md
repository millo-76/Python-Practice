# Baseball Stats Tracker ⚾

A simple Python project to record and view baseball player statistics — both with a console version and a GUI (using Tkinter).

## 📂 Features

- Add multiple batters
- Record stats: At Bats (AB), Hits (H), Walks (BB), and Runs
- Calculate batting average
- Save/load stats from a file (`player_stats.json`)
- Case-insensitive player name matching
- GUI version (if `tkinter` is installed)
